//七夕到了,这是我们在一起以后第一个关于情侣的节日.
//我们都是程序员,所以我写了一些代码送给你。希望你喜欢。
Boy i = new Boy ("步宇星");
Girl u = new Girl ("陈佩");
Couple we = new Couple (i,u);
// 2014年3月17日我第一次遇见你.感觉你是个非常开朗又讨人喜欢的女孩.
u.appear();
// 对你的印象很好,但当时我并没有想到你对我来说将会如此特别.
// 不过从那之后我就开始默默的关注你了.
i.careAbout(u);
// 一次偶然的机会和你在地铁上聊了很多,发现很多共同的爱好.
i.talkWith(u);
// 不经意间提到想去看电影.
i.say("想去看电影,又不知道该去哪看,有机会你带我去吧.");
// 原本只是随口说说,并没有抱有期待,没想到你送了我一张电影票.
u.giveMovieTicketTo(i);
i.dateWith(u); // 以此为机会,我们有了第一次约会.
i.like(u);// 期间非常愉快,我对你有了好感。
// 于是,五一请你当我的导游,你爽快的答应了.我们度过了悠闲舒适的一天.
u.guide(i);
we.haveANiceTime();
// 青年节那天你约我吃饭,说自己有私心,我偷偷的想难道你喜欢上我了,不会吧.
u.haveDinnerWith(i);
// 想不到你的私心竟然真的是向我表白！那一刻我简直开心坏了！
u.expressLoveTo(i);
// 虽然有些顾虑,但是得来不易的缘分不去珍惜,还有什么值得珍惜呢？
i.accept(u);
// 终于我们在一起了！彼此都不在是独自一人了！
we.failInLove();
we.areNotAlone();
// 相爱容易相处难,在一起的这些日子有苦有甜.
we.taste("sweet","bitter");
// 由于我内心的不安引起了很多不快,最终你让我下定决心去改变,去成长.
u.inspire(i);
i.wantGrowUpWith(u);
// 尽管我们有很多想法都不同,但我仍然相信我们就是注定在一起的.
// 希望我们以后一切都能顺顺利利,每一天都多爱一点点.
while (we.liveTegetherDays()) {
we.love.increase();
}
// 七夕快乐！
月月, 我们在一起已经
Love u forever and ever.
